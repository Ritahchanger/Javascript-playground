## ğŸš€ Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- Basic understanding of JavaScript

### Installation
```bash
# Clone the repository
git clone https://github.com/Ritahchanger/Javascript-playground.git

# Navigate to the project
cd Javascript-playground

# Install dependencies (for projects that require them)
npm install
```

### Running Examples
```bash
# Run individual JavaScript files
node classes/Inheritance.js

# For React projects
cd react-tuto
npm start

# For backend projects
cd backendjs
npm run dev
```
